# Installation

There are two ways to get Hurricane: through **Steam**, or **standalone** using the
Hurricane Updater.

## Option A - Steam (easiest)

Subscribe to the Hurricane item on the Steam Workshop, and Steam keeps it updated for you:

- [Hurricane on the Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3423755273)

Once subscribed, launch it from your Steam library.

## Option B - Standalone (Hurricane Updater)

Use the **Hurricane Updater** to download and keep the client up to date:

- [Download the latest Hurricane Updater](https://github.com/Nightdawg/Hurricane-Updater/releases/latest)

Run the updater, let it fetch the client, and you're ready to [launch it](./launching).

### If the updater doesn't work

1. Make sure your installed Java version is between **17 and 21** - **21 is highly recommended**.
2. You may need to add the updater file (`HurricaneUpdater.jar`) to your anti-virus
   exceptions list.

## Requirements

- **Java 17-21** (Java **21** is highly recommended). GraalVM 21 is also a great option and
  can give a noticeable FPS boost.
