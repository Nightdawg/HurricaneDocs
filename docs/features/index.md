# Features

Welcome to the Hurricane feature library. Each feature gets its own page with
screenshots, GIFs, and short videos that show exactly how it works.

> 🚧 This section is just getting started. Pages will be filled in over time.
> To add a new page, copy `docs/features/template.md` to a new file in
> `docs/features/` and write your content.

## Categories

These are the kinds of features Hurricane offers. Pages will be added under each over time.

- **Interface & HUD** - layout tweaks, widgets, and on-screen information.
- **Maps & Minimap** - custom markers, map-icon presets, private web-map server integration.
- **Quality of Life** - conveniences and automation that smooth out the grind.
- **Combat Helpers** - hitboxes, priority targets, and combat tooling.
- **Alarms & Notifications** - configurable sounds for animals, combat, and priority targets.
- **Integrations** - cookbook / food-book and private map server support.
