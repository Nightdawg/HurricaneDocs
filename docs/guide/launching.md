# Launching the Client

> This page applies to the **standalone** install. If you use Steam, just launch
> Hurricane from your Steam library.

## Windows

Run **`Play.bat`** inside the client folder.

## Linux / macOS

Run **`Play_Linux.sh`** inside the client folder.

## Java version

The client works with any Java version between **Java 17 and Java 21** -
**but Java 21 is highly recommended.**

::: tip GraalVM tip
GraalVM 21 (an open-source Java distribution based on OpenJDK) tends to give an extra
**15-20 FPS** out of the client. It's worth trying if you want more performance.
:::

## If the client doesn't launch

1. Make sure your installed Java version is between **17 and 21** - **21 is highly recommended**.
2. You may need to add the launcher file (`Play.bat` or `Play_Linux.sh`) to your anti-virus
   exceptions list.
