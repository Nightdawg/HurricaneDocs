## What is Hurricane?

**Hurricane** is a custom client you can use to play the wonderful game
[Haven & Hearth](https://www.havenandhearth.com/). It is built on top of the official
"Vanilla" client and does **not** depend on any other custom client. The base client's
changes from Loftar are merged in regularly to keep things up to date and avoid crashes.

You can play it **standalone** or **through Steam** via the Steam Workshop.

> **Privacy note:** This client does not send any data anywhere besides the official
> Seatribe server - unless you explicitly configure it to.
